算法题-矩阵排序
2018-12-18|algorithm

题目描述

编写函数
void order(int m, int n, int *matrix);
将 m × n 维的矩阵按照如下规则排序：
（1）首先将矩阵内每一列的元素值，按照从小到大的顺序排列；
（2）然后将矩阵各列按照该列元素之和的顺序从小到大排列，并将排序后的矩阵输出。
使用指针方法编程。

输入形式为：
m（行数 0 < m < 100）
n（列数 0 < n < 100）
矩阵元素 m 行 n 列（同一行的元素以空格隔开）

输出排序后的矩阵。若输入m,n不符合要求，则输出 “ERROR”。

输入

共 m+2 行整数：
m（行数 0 < m < 100）
n（列数 0 < n < 100）
矩阵元素 m 行 n 列（同一行的元素以空格隔开）

输出

排序后的矩阵元素 m 行 n 列（同一行的元素以空格隔开）

样例输入

3
3
1 2 9
4 8 6
7 5 3

样例输出

1 2 3
4 5 6
7 8 9
